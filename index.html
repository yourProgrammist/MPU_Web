NMAP
Сканирование IP-сетей

Утилита с открытым исходным кодом для исследования сети и проверки безопасности.

Сканирование
больших сетей

Nmap использует "сырые" IP пакеты оригинальным способом, чтобы определить какие хосты доступны в сети, какие службы (название приложения и версию) они предлагают, какие операционные системы (и версии ОС) они используют, какие типы пакетных фильтров/брандмауэров используются и еще множество других характеристик.
Для кого
Пентестер или интересуешься
информационной безопасностью? Тебе будет
актуально
  • Сканер с минимальным уровнем порога знаний
  • Наиболее простой синтаксис среди аналогов 
    nmap [ <Тип сканирования> ...] [ <Опции> ] { <цель сканирования> }
  • Большое количество флагов для самых разных целей
  • Nmap поддерживает множество различных типов сканирования, включая сканирование TCP/UDP, обнаружение версий сервисов, сканирование операционных систем и даже скриптовые проверки безопасности.
  • Nmap способен обнаруживать активные устройства в сети, их IP-адреса, используемые ими сервисы и открытые порты, что делает его незаменимым инструментом для администраторов сетей и специалистов по информационной безопасности.
  • Преимущества nmap перед аналогами
  • Nmap доступен на множестве операционных систем, включая Windows, Linux и macOS, что делает его доступным для широкого круга пользователей;
  • Nmap имеет систему скриптов на языке Lua, которая позволяет пользователям создавать свои собственные скрипты для автоматизации задач и проведения более сложных проверок безопасности;
  • У Nmap есть обширная документация, а также большое и активное сообщество пользователей и разработчиков, которые могут предложить поддержку и руководство;
  • Nmap является бесплатным и открытым, что позволяет пользователям изучать его исходный код и адаптировать инструмент под свои нужды
